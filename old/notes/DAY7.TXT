Simple Git Commands:
1. git init // initializes a given folder as git repo, only run once in the beginning
2. git status // gives an overview of what files are staged or unstaged
3. git add <filesnames> // to stage or track the changes before commtiing
4. git commit -m "<message>" // finalizes the changes and creates a commit id to track the changes
5. git push origin <branch name> // to push your local repo to central
6. git pull branch name // pull a workspace or changes from central to local
7. git fetch // fetch or pull metadata of other branches from central to local
8. git merge // combine two branches changes into single branch
9. git log // gives all commit information or history
10. git log --oneline // gives all the history in single line with commit id and message
11. git show <commit id> // shows complete info of that commit id or all the changes at that point of time
